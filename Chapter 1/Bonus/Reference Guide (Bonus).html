HTML & CSS Elements Reference Guide

HTML Elements

<<>!DOCTYPE> - Used to define the type of document.

<html> - Used to define that the document is HTML.

<head> - Used to contain the metadata of the dcoument and webpage.

<title> - Used to name the webpage.

<body> - Used to contain the contents of the webpage.

<h1> to <h6> - Used to create titles for specific sections.

<p> - Used to insert a paragraph.

<div> - Used to define a division or section within a webpage.

<ol> - Used to define an ordered list, typically shown in numbers.

<ul> - Used to define an unordered list, typically shown in bullet points.

CSS Elements

There are 3 ways to insert CSS: inline, internal, and external.

Inline Styling - Uses CSS properties within the HTML document.

 
            <!DOCTYPE html>
            <html>
            <body>
    
            <h1 style="color:blue;text-align:center;">This is a heading</h1>
            <p style="color:red;">This is a paragraph.</p>
                    
            </body>
            </html>

Internal Styling - Uses CSS properties within the HTML document. 


            <!DOCTYPE html>
            <html>
            <style>
            body {
                background-color: linen;
            } 
            h1 {
                color: maroon;
                margin-left: 40px;
            }
            </style>
            <body>
            
            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>

            </body>
            </html>

External Styling - Uses CSS properties by linking a CSS document. 


            <!DOCTYPE html>
            <html>
            <head>
            <link rel="stylesheet" href="mystyle.css">
            </head>
            <body>

            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>

            </body>
            </html>

Color - Can be used to define the color of parts of various elements.


            h1 {
                color:DodgerBlue;
            }
            p {
                color:Tomato;
            }

Margin - Used to create space around elements.


            p {
                margin: 100px 120px 120px 100px;
            }

Padding - Used to create space around an elements content and the border.


            p {
                padding: 50px 30px 50px 80px;
            }

Border - Used to define the border of an element.


            p {
                border: 20px 10px 10px 20px;
            }

Font - Used to define the font of the text.


            p {
                font-family: "Times New Roman", Times, serif;
            }

Align - Used to align the elements of a webpage.


            div {
                display: flex;
                justify-content: center;
                align-items: center;
                height: 200px;
                border: 3px solid green;
            }
